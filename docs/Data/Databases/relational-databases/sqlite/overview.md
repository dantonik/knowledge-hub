# SQLie

In-process SQL database engine.

## Commands
`.quit`  
`.tables`

#### Modes/ Isolation Levels
- Deferred
- Immediate
- Exclusive

#### Locks
- Shared Lock
- Reserved Lock
- Pending Lock
- Exclusive Lock
[SQLite locking](https://www.sqlite.org/lockingv3.html)

WAL mode requires SQLite 3.7.0
PRAGMA journal_mode = WAL
