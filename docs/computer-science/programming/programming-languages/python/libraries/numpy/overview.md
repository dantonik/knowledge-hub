Numeric Python

Main feature: Arrays  
It's the basis for SciPy and Pandas.  

## N-D array class
N-Dimensional array.  
- 0th -> point
- 1st -> line
- 2nd -> plane

`[1, 2, 3, 4]` like Python's list but operations work elementwise.  

```
import numpy as np
```
